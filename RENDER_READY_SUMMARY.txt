================================================================================
WHAT HAPPENS ON DEPLOY
================================================================================

Build Phase (Render runs):
  1. Clone repository
  2. npm install (installs root + server deps)
  3. npm run build:frontend (Vite build)
  4. npm run build:backend (TypeScript compilation)
  5. Creates optimized bundle (~200MB)

Release Phase (Procfile):
  1. Runs migrations: cd server && npm run migrate:prod
  2. Prepares database schema
  3. Ensures custom items support enabled

Start Phase:
  1. node server/dist/index.js
  2. Connects to PostgreSQL
  3. Initializes WebSocket service
  4. Serves frontend from dist/client
  5. Listens on port 10000
